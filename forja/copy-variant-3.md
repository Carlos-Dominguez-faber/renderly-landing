# Copy Variant 3: Outcome-Focused

> This copy replaces the default Renderly copy for Sandbox 3 (Disruptivo).
> The angle is **results-driven + emotional storytelling + local-first**.
> Instead of describing what the product does, it describes what the user GETS.

---

## How to Use

Sandbox 3 should create a NEW `copy.ts` (or modify the existing one) using the
EN content below. The ES translation should be adapted to match the same tone.
The structure/interface stays the same — only the text values change.
